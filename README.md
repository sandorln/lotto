# 로또 합니다

동행복권에서 제공하는 API를 통해,
최신 회차 ~ 이전 회차의 당첨 번호 및 정보를 확인할 수 있습니다
로또 번호를 무작위로 생성합니다.

## 개발 환경
- Kotlin, MVI, Compose, Hilt, Coroutine, Retrofit2, Navigation Component
- Min SDK 21
